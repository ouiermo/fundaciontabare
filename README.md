Fundación Tabare - Plataforma de Salud Mental Integral
"Abrazamos tu historia para fortalecer la esperanza."

Este proyecto es la cara digital de la Fundación Tabare, una red multidisciplinaria que ofrece apoyo integral en las áreas psicológica, social, legal, terapéutica y educativa. El sitio ha sido diseñado bajo un modelo de "Sentido Social", enfocándose en la accesibilidad y el bienestar como un derecho, no un privilegio.

🛠️ Tecnologías Utilizadas
El sitio fue desarrollado e implementado utilizando un stack moderno para garantizar rendimiento, SEO y una experiencia de usuario fluida:

Framework: React + Vite

Estilos: Tailwind CSS

Componentes: Shadcn/UI

Iconografía: Lucide Icons

Despliegue: Lovable / GitHub Pages / Netlify

🎨 Identidad Visual y Tono
El diseño sigue estrictamente los requerimientos de marca de la Fundación:

Paleta de Colores: * Primario: Amarillo (Luz y esperanza).

Base: Blanco y Gris Claro (Calma y limpieza).

Tono de Voz: Cercano, amable y profesional.

Filosofía de Contenido: Se evita estrictamente el término "Gratis". En su lugar, se comunica "Accesibilidad", "Compromiso" y "Sostenibilidad Social" para fomentar un rol activo del usuario en su proceso.

📂 Estructura del Proyecto
src/components/: Componentes reutilizables (Botones, Cards, Navbar, Footer).

src/pages/: Vistas principales (Home, Blog, etc.).

public/: Archivos estáticos, incluyendo los Favicons actualizados.

content/ (o similar): Archivos Markdown para la gestión simplificada del blog.

🚀 Instalación y Desarrollo Local
Si deseas realizar cambios locales o extender la funcionalidad, sigue estos pasos:

Clonar el repositorio:

Bash

git clone https://github.com/tu-usuario/fundacion-tabare.git
Instalar dependencias:

Bash

npm install
Iniciar servidor de desarrollo:

Bash

npm run dev
Construir para producción:

Bash

npm run build
